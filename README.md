# AI-Fill

AI-Fill is an advanced, automated form-filling solution designed to streamline job applications with unmatched precision and speed. By leveraging state-of-the-art Large Language Models and intelligent browser automation, AI-Fill eliminates the repetitive nature of manual data entry, allowing users to focus on what truly matters: securing their next career move.

![AI-Fill Landing Page](Photos/Screenshot%202026-03-28%20at%209.43.55%E2%80%AFAM.png)

## Overview

In a competitive job market, agility is a requirement. AI-Fill provides a sound and reliable framework that adapts to diverse application platforms including Workday, Greenhouse, Lever, and ICIMS. It parses complex form structures in real-time and maps them against your professional profile with extreme accuracy.

## Automation in Action

### Intelligent Execution
Launch applications by simply providing a URL. AI-Fill handles the navigation and field mapping, providing real-time execution logs for full transparency.

![Launch AutoFill Phase 1](Photos/Screenshot%202026-03-28%20at%209.44.16%E2%80%AFAM.png)

### Multi-Stage AI Validation
Every application is reviewed by a secondary AI Validator. This "AI Critic" pass audits the initial results, filling in gaps and correcting discrepancies before you ever hit submit. This dual-layered approach ensures the highest level of reliability.

## Technology Stack

AI-Fill is built on a high-performance stack for maximum efficiency:

- Backend: Python with Playwright for robust browser orchestration.
- Frontend: React-based interface for a smooth, modern user experience.
- Intelligence: Integration with OpenAI and Ollama (local LLMs) for adaptive decision making.
- Automation: Advanced JS-based field extraction and human-mimicry algorithms.

## Getting Started

1. Set up your professional profile in the Profile Setup section.
2. Navigate to Launch AutoFill.
3. Paste the job application URL.
4. Watch as AI-Fill handles the heavy lifting, then review and submit.

AI-Fill is the agile partner you need to scale your job search and reclaim your time.
